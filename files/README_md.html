# Otto - 0.2

**Auto-define your rack-apps in plaintext.**

## Installation

Get it in one of the following ways: 

$ gem install otto
$ sudo gem install otto
$ git clone git://github.com/delano/otto.git

You can also download via [tarball](github.com/delano/otto/tarball/latest) or [zip](github.com/delano/otto/zipball/latest).
